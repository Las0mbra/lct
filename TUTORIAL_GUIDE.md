# LCT 40K Table: Quick Guide

LCT is a ready-to-play Warhammer 40,000 table for **Tabletop Simulator (TTS)**. It provides the battlefield, mission tools, dice rollers, score tracking, and measuring aids needed to run a game online.

You still provide your own army models and play the rules yourself; the table handles much of the setup and bookkeeping around the game.

For detailed instructions on the table's more advanced controls, see the [Advanced Feature Guide](TUTORIAL_Advanced.md).

## Current Features

### Game Setup

- **Red, Blue, or Spectator seats** - Choose a player side or watch the game.
- **1v1 and Solo modes** - Set up a standard match or use the table alone.
- **Multiple battlefield sizes** - Includes Combat Patrol, Incursion, Strike Force, Onslaught, Boarding Actions, and King of the Colosseum sizes.
- **Mission pack selection** - Play using the included Pariah Nexus mission content.
- **Disposition selection** - Each player selects a disposition, which is used to prepare suitable primary missions and deployment choices.
- **Mission generation** - Deals each player's primary mission and presents matching deployment cards.
- **Deployment selection** - Pick from the offered deployment layouts or choose one randomly.
- **Automatic deployment zones** - The selected deployment zone is drawn on the battlefield and set automatically when a map is loaded.
- **Automatic objective and ruin markers** - Markers are placed according to the selected layout and deployment zone.
- **Map loading tools** - Load included terrain layouts and reposition saved groups of map objects.
- **Army placement tools** - Place an army for Red or Blue, hide it during setup, reveal it, or remove it.
- **Table appearance controls** - Choose a green, blue, grey, or purple mat, or apply and save custom mat/table images.

### Missions and Scoring

- **Tactical or Fixed secondaries** - Choose a shuffled Tactical deck or select exactly two Fixed secondary missions.
- **Secondary card controls** - Draw one or two cards, discard cards, and recycle cards back into the deck.
- **GT scoresheet** - Optional automated score sheet with primary, secondary, and total score limits.
- **VP and CP overlay** - Both players can see Victory Points (VP), Command Points (CP), turns, phases, and the current battle round.
- **Quick overlay controls** - Pass the turn, gain CP, or spend CP directly from the overlay.
- **Detailed scoreboard** - Shows round-by-round scoring plus active, discarded, and remaining secondary missions.
- **Minimalist UI mode** - Collapse the overlay when more screen space is needed.
- **Spectator overlay** - Spectators can follow both players' scores, turns, phases, and clocks.
- **Turn and phase tracking** - Tracks Command, Movement, Shooting, Charge, and Fight phases.
- **Round and game-end tracking** - Advances battle rounds and can announce the winner or a draw.
- **Force Start** - Press Start three times to bypass incomplete Tactical/Fixed secondary selection when needed.

### Dice Tools

- **Personal dice mats** - Red and Blue each have a dedicated rolling area.
- **Quick dice spawning** - Add common groups of D6 dice or a D3.
- **Roll, select, sort, and reroll controls** - Manage all dice or individual result rows.
- **Ordered roll groups** - Roll dice in separate ordered groups when attacks must be resolved in sequence.
- **Lethal and Sustained Hits tools** - Automatically separate Lethal Hits, duplicate Sustained Hits, or apply both together; supports 5+ and 6+ thresholds.
- **Custom dice support** - Use custom dice designs and switch pip colours.
- **Copy opponent dice** - Copy the other player's current dice pool.
- **Undo recent roll** - Revert a roll before it locks in.
- **Roll history** - View the last roll or a summary of the last five rolls.
- **Clear controls** - Remove a row of dice or clear the full dice mat.

### Measuring and Battlefield Aids

- **Measurement bubbles** - Toggle 1", 2", 3", 6", 8", 9", 12", or 15" ranges around selected models.
- **Custom measurement bubble** - Enter a custom range using the table UI or a hotkey.
- **Clear bubbles and drawn lines separately** - Remove measurement aids without clearing everything.
- **Automatic model spacing** - Arrange selected models with 1" or 2" gaps.
- **Live coherency check** - Shows distances between selected models and highlights models outside 2" coherency or more than 9" apart.
- **Line-of-sight markers** - Place player-specific markers to help check visibility.
- **Deployment zone display** - Show or hide the active deployment areas.
- **Objective display** - Show objective locations and ranges.
- **Territory and area guides** - Display territory lines, table quarters, area-denial ranges, and Strategic Reserves boundaries.
- **Save and restore positions** - Temporarily save selected model positions and return them later.

### Models, Tokens, and References

- **Wound counters** - Spawn counters, link them to models, and adjust remaining wounds.
- **Activation/status tokens** - Spawn and link status markers to units.
- **CP counters** - Physical counters stay synchronized with CP changes made through the overlay.
- **Reserves boards** - Moveable off-table areas for units in reserve.
- **Token collection** - Includes additional gameplay tokens and a memory bag for reusable object sets.
- **Stat helpers** - Resolve groups of weapon attacks with configurable dice and spawn positions.
- **Rules references** - Includes bookmarked Core Rules and a Stratagem reference image for in-game use.
- **Useful hotkeys** - Bind shortcuts for rolling dice, clearing dice or measurements, showing the scoreboard, toggling the UI, saving positions, restoring positions, spacing models, and more.
